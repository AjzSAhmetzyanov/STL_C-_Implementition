В рамках данного проекта реализована собственная библиотека, основных стандартных контейнеров класса языка С++: list (список), map (словарь), queue (очередь), set (множество), stack (стек) и vector (вектор). Реализация предоставляет весь набор стандартных методов и атрибутов для работы с элементами, проверкой заполненности контейнера и итерирования. В качестве дополнительного задания реализовано еще несколько не так часто используемых, но отличающихся деталями реализации контейнерных классов из контейнерной библиотеки C++.
